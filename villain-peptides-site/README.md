# Villain Peptides Website

Research-only React/Vite website.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy on Vercel
1. Create a GitHub repo.
2. Upload these files.
3. Go to Vercel.
4. Import the GitHub repo.
5. Click Deploy.

## Important
This template uses research-only language and an order-request checkout section.
Before accepting payments, get legal/compliance review and confirm your payment processor allows your business category.
